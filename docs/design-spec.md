# 设计规范 — 波动贡献分析工具

## 色彩

| 用途 | 颜色 |
|------|------|
| 主色 | `#2563EB` (蓝) |
| 背景 | `#F8FAFC` |
| 卡片 | `#FFFFFF` |
| 文字 | `#1E293B` |
| 辅助文字 | `#64748B` |
| 增加 (delta+) | `#16A34A` |
| 减少 (delta-) | `#DC2626` |

## 布局

- 最大宽度 1200px，居中
- 三步骤流程：上传 → 配置 → 结果
- 移动端响应式

## 组件规范

### 上传区
- 拖拽上传区域，虚线边框
- 支持点击选择文件
- 上传后显示：文件名 + 行数 + 列数 + 前 5 行预览

### 配置面板
- 下拉选择器 × 5（维度可多选）
- 时间字段选择后，输入基期/本期标签
- "开始分析"按钮

### 结果表
- 横向滚动表格
- 数值保留 4 位小数
- delta 列正负颜色区分
- 底部汇总行粗体
- 贡献度列显示百分比
